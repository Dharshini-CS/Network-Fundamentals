
# What is Network ?

 A group of interconnected device 

## Networking tools :

- **Hub** : Network device that broadcasts information to every device on the network.
- **Switch** :Connects devices in a network and forwards data only to the specific device (using MAC addresses). Provides multiple Ethernet ports for connecting many devices. 
- **Router**:Connects different networks (like your LAN to the internet). It directs traffic between them and often provides Wi‑Fi.
- **Modem**:  Converts your ISP’s signal (fiber, DSL, cable) into usable internet and connects to the router, bringing internet access to your LAN.
